2025 NUEDC 15F题：高精度数字信号参数测量系统 (FPGA)本项目是针对全国大学生电子设计竞赛（NUEDC）15F题开发的 FPGA 核心测量系统。系统基于 Xilinx Vivado 开发环境，利用 FPGA 高速并行的特性，实现了对信号频率、周期、占空比以及两路信号间时间间隔的高精度实时测量。🌟 系统核心特性采样主频：通过 MMCM 倍频至 400MHz，理论时间分辨率达 2.5ns。等精度测量：频率测量采用等精度闸门技术，确保在不同频段下均有稳定的测量精度。稳定算法：所有输入信号经过三级同步处理，有效消除亚稳态，提高系统抗干扰能力。自动量程：顶层模块根据测量结果自动切换频率/周期显示模式，优化用户体验。📂 目录结构说明为了方便维护，建议保持以下文件结构：.

🛠 模块功能简述1. 顶层调度 (Top_Module.v)系统的核心大脑。它负责例化时钟管理 IP (MMCM)、各子测量模块，并根据信号特征实时调度输出数据：当周期小于阈值 (6000 ticks) 时，系统认为信号频率较高，优先输出频率信息。当周期较大时，系统切换模式，输出带标记的周期信息。2. 频率测量 (Freq_Measure.v)原理：采用 1 秒标准闸门计数法。功能：精确捕捉信号上升沿并计数，适合测量高频信号。3. 占空比与电平计数 (duty.v)功能：测量高电平与低电平持续的时钟周期数。特性：包含超时自动归零逻辑，能够正确识别 0% 和 100% 占空比。4. 时间间隔测量 (Time_Interval_Measure.v)功能：测量两路输入信号之间的相对时间延迟。实现：通过边沿检测产生测量脉冲，并复用计数逻辑进行高精度计时。🚀 快速开始导入工程：将本仓库中的源码文件添加至你的 Vivado 工程。配置 IP 核：确保工程中包含名为 clk_wiz_0 的时钟 IP，输出频率设置为 400MHz。约束引脚：编辑 .xdc 文件，将信号输入与板载引脚对应。编译下载：依次运行 Synthesis -> Implementation -> Generate Bitstream。本项目仅供电赛学习与交流使用。
